Information de la machine à sous Olympique : 

WINDOWS | LINUX

Touches de contrôle et de raccourci :

	* Machine à sous :
		- Bouton SPIN => ESPACE
		- Bouton MINI-JEUX => DOLLAR
		- Bouton ? => EXCLAMATION
		- Bouton MAX => RIGHT
		- Bouton ^ => UP
		- Bouton v => DOWN

	* Dans les menus : 
		- Bouton <- => LEFT

	* Jeu 1 | Saut de haie :
		- Saut => ^

	* Jeu 2 | Escalade :
		- Saut => ESPACE
		- Droite => RIGHT
		- Gauche => LEFT

	* Jeu 3 | Athlétisme :
		- Courir => ESPACE

	* Jeu 4 | Natation :
		- Bras Droit => RIGHT
		- Bras Gauche => LEFT
